# Student Management System

A comprehensive Student Management System developed using Python Flask and Object-Oriented Programming (OOP) principles. 
The application provides an efficient platform for managing student information, course details, enrollments, and academic operations 
through a user-friendly web interface.

## Features

* Secure Admin Authentication
* Student Registration and Management
* Course Management
* Student Enrollment System
* Dashboard with Academic Statistics
* CRUD Operations for Students and Courses
* SQLite Database Integration
* Modular Flask Architecture
* Responsive User Interface
* Service Layer Implementation
* Clean OOP-Based Design

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript
* SQLite
* Jinja2 Templates
* Object-Oriented Programming (OOP)

## Project Architecture

The project follows a layered architecture:

Routes → Services → Models → Database

This structure improves code maintainability, scalability, and separation of concerns.

## OOP Concepts Implemented

* Classes and Objects
* Encapsulation
* Abstraction
* Modular Design Principles
* Service-Oriented Architecture

## Learning Outcomes

This project demonstrates practical implementation of Flask web development, database management, CRUD operations, authentication systems, and Object-Oriented Programming concepts in a real-world academic management application.
